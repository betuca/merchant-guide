Informações sobre o exercício Merchant's Guide to the Galaxy


Informações básicas

A soluçao do problema do mercante das galáxias foi feita em Java utilizando um Desenvolvimento Orientado por Testes e tendo em mente princípios do Desenvolvimento Orientado a Objetos. Cada classe foi criada com a intenção de ter uma função claramente definida e de ser o mais simples possível.


Pré-requisitos

  • Para compilar e executar o projeto é necessário utilizar o JDK 1.8

  • As instruões de execução desde arquivo consideram que o Maven já está instalado e configurado no path do sistema

  • Alguns dos comandos deste arquivo consideram que o java está configurado no path


Informações para execução

Para facilitar a compilacão e execução do código o arquivo pom.xml se encontra no arquivo compactado.

Para compilar o projeto (estando no diretório do arquivo pom.xml):

mvn compile

Para executar a classe principal (estando no diretório do arquivo pom.xml):

java -cp target/classes exercise.MerchantsUnitConverter caminho_para_arquivo_de_entrada

Preenchendo o "caminho_para_arquivo_de_entrada" com o caminho completo ou relativo do arquivo de entrada.

O programa apresentará as respostas as perguntas do arquivo de entrada no console do sistema.


Testes sugeridos na questão

Para ver a aplicação funcionando com o teste de entrada sugerido no e-mail, existe um arquivo nos resources do projeto que representa precisamente a entrada sugerida. Para ver a saída dada pelo sistema basta executar o comando:

java -cp target/classes exercise.MerchantsUnitConverter src\main\resources\suggested-test.txt

E observar a saída no console.


Suposições

Como não há detalhes suficientes na descrição alguns comportamentos de erro foram definidos para a entrada

  • O arquivo de entrada deve seguir uma estrutura similar a estrutura dada no exemplo do exercício.

  • O arquivo de entrada deve possuir todas as “seções” encontradas no exemplo do exercício (definições de valores, definições de conversões dos metais e perguntas).

  • Exceções são lançadas em casos de formatos desconhecidos e a execução é interrompida.
